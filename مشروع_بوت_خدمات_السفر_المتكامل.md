# مشروع بوت خدمات السفر المتكامل

هذا المشروع عبارة عن بوت تيليجرام متكامل مصمم لتقديم خدمات السفر، مع دعم الردود المحلية والذكاء الاصطناعي، وجاهز للنشر على منصة Render.

## هيكل الملفات

| الملف | الوصف |
| :--- | :--- |
| `bot.py` | الملف الرئيسي للبوت، يحتوي على المنطق البرمجي وإعدادات Flask وWebhook. |
| `requirements.txt` | قائمة المكتبات المطلوبة لتشغيل المشروع. |
| `Procfile` | ملف إعدادات التشغيل لمنصة Render. |
| `render.yaml` | ملف تكوين الخدمات على Render لسهولة النشر. |
| `.env.example` | نموذج للمتغيرات البيئية المطلوبة (Token البوت ومفتاح الذكاء الاصطناعي). |

## الميزات الرئيسية

1.  **دعم Webhook:** تم إعداد البوت ليعمل عبر Webhook، وهو الخيار الأمثل للمنصات السحابية مثل Render لضمان استجابة سريعة وتوفير الموارد.
2.  **الردود المحلية:** يحتوي البوت على نظام ردود محلية للأسئلة الشائعة (مرحبا، خدماتكم، إلخ).
3.  **الذكاء الاصطناعي:** تم دمج **Google Gemini AI** لتوفير ردود ذكية وشاملة في حال لم يتوفر رد محلي.
4.  **التوافق مع Render:** جميع الملفات مهيأة للعمل مباشرة على Render دون الحاجة لتعديلات إضافية.

## كيفية الإعداد والنشر

### 1. إعداد المتغيرات البيئية
يجب إضافة المتغيرات التالية في إعدادات Render (Environment Variables):
- `TELEGRAM_BOT_TOKEN`: التوكن الخاص ببوتك من @BotFather.
- `GEMINI_API_KEY`: مفتاح API من Google AI Studio (اختياري لتفعيل الذكاء الاصطناعي).

### 2. تفعيل الـ Webhook
بعد تشغيل البوت على Render، ستحتاج لضبط رابط الـ Webhook الخاص بتيليجرام ليشير إلى رابط تطبيقك:
`https://api.telegram.org/bot<YOUR_BOT_TOKEN>/setWebhook?url=https://<YOUR_APP_NAME>.onrender.com/webhook`

## ملاحظات هامة
- تم ضبط العملة الافتراضية للخدمات لتكون **الريال السعودي (SAR)** بناءً على أفضل الممارسات لخدمات السفر في المنطقة.
- البوت يستخدم مكتبة `python-telegram-bot` بإصدارها الأحدث (21.4) لضمان التوافق والأمان.
